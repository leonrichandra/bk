# Workshop Data Science: Mengolah Data Menjadi Cerita Bisnis

Workshop ini dibuat untuk siswa/i SMAN 1 Boyolali sebagai pengenalan praktis Data Science menggunakan Python dan Google Colab.

## Tujuan Workshop

Peserta akan belajar bagaimana mengubah data mentah menjadi laporan bisnis sederhana yang rapi, informatif, dan mudah dipahami.

Studi kasus yang digunakan adalah analisis penjualan produk susu di Boyolali.

## Tools yang Digunakan

- Google Colab
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Scikit-learn

## File yang Tersedia

- `dataset_penjualan_susu_boyolali.csv`  
  Dataset utama yang digunakan dalam workshop.

- `student_blank_colab_workshop_boyolali.ipynb`  
  Notebook untuk peserta. Peserta akan mengetik dan menjalankan kode secara bertahap.

- `facilitator_complete_colab_workshop_boyolali.ipynb`  
  Notebook lengkap untuk fasilitator, berisi seluruh kode, komentar, analisis, visualisasi, dan model prediksi.

- `workshop_data_science_boyolali.pptx`  
  Slide pembukaan dan pengantar workshop.

## Cara Menggunakan Notebook

1. Buka Google Colab:  
   https://colab.research.google.com

2. Pilih:
   `File` → `Open Notebook` → `GitHub`

3. Masukkan link repository GitHub ini:
   `https://github.com/leonrichandra/bk`

4. Pilih notebook yang ingin digunakan.

5. Jalankan kode dari atas ke bawah sesuai arahan fasilitator.
